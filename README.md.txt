# 🏠 House Price Prediction using Machine Learning

## 📌 Project Description
This project predicts house prices using Machine Learning techniques.
The model is trained on housing dataset using Linear Regression.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📂 Dataset
The dataset contains various features such as:
- Area
- Bedrooms
- Bathrooms
- Stories
- Parking
- Price (Target variable)

## ⚙️ Steps Performed
1. Data Preprocessing
2. Exploratory Data Analysis
3. Model Training
4. Model Evaluation

## 📈 Model Used
Linear Regression

## 🎯 Result
The model predicts house prices with good accuracy.
